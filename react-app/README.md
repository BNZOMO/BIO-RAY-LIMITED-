React scaffold for BR Shield — Vite + Framer Motion + Lottie + GSAP

Install:

```bash
cd react-app
npm install
# or yarn
```

Run dev:

```bash
npm run dev
```

Notes:
- Lottie JSON path in `Hero` uses a public sample; replace with your JSON or local asset.
- Particles run only on desktop and respect `prefers-reduced-motion`.
- GSAP ScrollTrigger is used in `Projects` — ensure `npm install gsap`.
